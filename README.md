# FocusFlow
**Course:** User Interface Design (End-Semester Project)
**Tech Stack:** HTML5, CSS3, Vanilla JavaScript (Frontend Only)

## Project Description
FocusFlow is an interactive, frontend-only productivity dashboard designed to improve focus and encourage consistency. Inspired by aesthetic, immersive digital workspaces, it uses gamification techniques such as focus scores, streak tracking, and a "Flow State" environment to keep users engaged. It includes a custom priority-based task manager, a Pomodoro focus lab, and dynamic progress visualization. All data is managed locally via the browser's localStorage.
